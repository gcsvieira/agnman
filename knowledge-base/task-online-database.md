# Task: Online Resource Database

## Description
Design and implement a system to download skills and rules from a remote repository or server.

## Requirements
- Define a remote source format (e.g., Git repository, ZIP archives).
- Implement `agnman skill/rule download <name>`.
- Handle versioning or updates of downloaded resources.
- Ensure security and validation of remote content.
